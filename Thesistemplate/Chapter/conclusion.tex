\chapter{Conclusion}
\label{ch:conclusion}

This thesis presents the design and implementation of a web-based vulnerability scanning platform that detects security flaws in Python applications using two complementary approaches: static code analysis and machine learning--based detection. The platform targets four critical web vulnerabilities---SQL injection, Cross-Site Scripting (XSS), command injection, and Cross-Site Request Forgery (CSRF)---and integrates the BiLSTM model from Farasat and Posegga~\cite{Farasat2024MLPython}, which is evaluated on seven vulnerability types (including remote code execution, path disclosure, and open redirect) on the VulnerabilityDetection dataset. Chapter~\ref{ch:results} (Results and Comparison) presents per-vulnerability comparisons between the static analyzer, the ML detector, and Semgrep for all four vulnerability types.
