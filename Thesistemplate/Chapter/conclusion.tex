\chapter{Conclusion}
\label{ch:conclusion}

This thesis presented a web-based vulnerability scanning platform for Python applications that combines a static analysis and a machine learning--based scanner. It enables flexible code analysis through direct input, file uploads, and GitHub repository scanning, providing an accessible alternative to commercial tools. The system detects key vulnerabilities---including SQL injection, XSS, command injection, and CSRF---using both AST-based static analysis and a BiLSTM model. A comparative evaluation demonstrates the strengths of each approach, while the platform's scalability makes it a practical tool for developers and researchers. This vulnerability scanner is opensource and is available at: https://github.com/mohammadalmasi/06.27
